伝達事項

    編集方法

        編集するべきフォルダ
            ./data
            - ./src
        
        毎年すること
            「./data/member」に一回生の分のデータを追加
            「./data/result」にリザルトのデータを追加
            「./data/schedule-data.js」のデータを更新
            「./data/greeting-data.js」のデータを更新
            「./src/greeting」で写真を変更
            「./src/member」で一回生の写真を追加
            「./src/result」でリザルトの写真を追加
            「./data/year-data.js」の更新

        たまにすること
            マニュアルが追加されるとき
            「./data/manual-data.js」に追加
            「./src/manual」にデータを追加
            スポンサーが追加されるとき
            「./data/sponsers-data.js」に追加
            「./src/sponsers」にデータを追加
            関連団体が追加されるとき
            「./data/support-data.js」に追加
            「./src/support」に写真を追加

    注意事項
        greetingの主将と前主将の写真は、縦：横＝2：3に加工が必要
        （20??data.jsが作られていないことで）ファイル読み込みに失敗すると、他の部分も表示されない可能性がある。
        instagramの写真は編集しなくても良いと思う。
